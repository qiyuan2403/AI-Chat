以下为更改内容

内置可视化控制台：可在界面中实时查看日志、错误和网络请求详情，支持导出日志文件
· API 提供商管理：支持添加、编辑、删除多个 API 提供商，并可随时切换当前使用的服务商。
· 可视化流式输出开关：可在设置中控制是否逐字显示 AI 生成过程，关闭后将一次性展示完整回复

性能优化

· 移除所有动画中的模糊滤镜，大幅降低渲染开销，界面滚动和流式输出更加流畅
· 对话切换时采用分片渲染，批量加载历史消息时不会阻塞界面操作
· 为对话列表项添加布局隔离属性，减少重绘范围
· 滚动事件增加节流处理，避免高频触发导致卡顿

交互改进

· 移动端键盘弹起时视图自适应更精准，输入框不会被遮挡
· 桌面端侧边栏与右侧预览面板的开关联动逻辑优化，避免空间挤压
· 模型选择器预设按钮与设置中的模型绑定保持同步，切换更符合直觉
